# architect-serverless-node

architect-serverless-node

## Requirement

- install yarn
- install node (v12+)
- install architect <https://github.com/architect/architect>

## Testing and run

```zsh
// test api in local
$ yarn run dev

// deploy to aws
$ yarn run deploy

// export sam.json
$ yarn run package

// set environment variables
$ yarn run env

// remove serverless services in aws
$ yarn run remove
```
